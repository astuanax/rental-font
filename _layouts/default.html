The Rental-Icon is growing into a set of icons specific to the rental industry. The icon provides an overview of room facilities, sports, events and rental company icons.

Time saver

The Rental-Icon is filling up a void. Designers and developers are uing a myriad of PNG images with custom icons. The Rental-Icon font can be styled with whatever CSS you can come up with.

Get Started

Setting up Rental-Icon can be as simple as adding two lines of code to your website!

  • Download and copy the entire rental-icons directory into your project.
  • In the <head> of your html, reference the location to your rental-icon.min.css. 
    
    <link rel="stylesheet" 
        href="path/to/font-awesome/css/rental-icon.min.css">
